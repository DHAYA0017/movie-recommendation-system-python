# Movie Recommendation System using Python

## Overview
An interactive command-line movie recommendation system built using Python.

The system provides genre-based movie recommendations, allows users to search for movies, and supports adding new movies dynamically.

---

## Features

- View available movie genres
- Get random movie recommendations
- Search movies by name
- Add new movies to the database
- Interactive menu-driven interface

---

## Technologies Used

- Python
- Random module
- Lists
- Dictionaries
- Functions

---

## How to Run

Clone the repository:

```bash
git clone https://github.com/DHAYA0017/movie-recommendation-system-python.git
```

Run the program:

```bash
python main.py
```
